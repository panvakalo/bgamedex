# PR Ready

Prepare the PR by running cleanup and commit workflows in sequence.

## Instructions

Execute these commands in order, waiting for each to complete before starting the next:

1. **Run `/clean-tests`** - Remove low-value tests from test files
2. **Run `/clean-comments`** - Remove redundant comments from code
3. **Run `/git-commit`** - Create a conventional commit with the changes

Wait for each command to fully complete before proceeding to the next one.
